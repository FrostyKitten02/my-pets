Posebnosti Strežnika

Tehnične zahteve za implementacijo strežnika vključujejo naslednje:

Tehnološki Stack

  • Programski jezik: JavaScript (Node.js)
  • Okvir: Express.js (za enostavno upravljanje HTTP zahtev)
  • Baza podatkov: PostgreSQL (za trajno shranjevanje podatkov)
  • Avtentikacija: JWT (JSON Web Token) za prijavo in avtorizacijo uporabnikov
  • API komunikacija: RESTful API za izmenjavo podatkov s klientom

Varnostne Zahteve

  • Uporaba HTTPS za šifrirano komunikacijo.
  • Omejitev dostopa do občutljivih podatkov glede na uporabniške pravice.

Sinhronizacija Podatkov

Podatki o živalih, prigrizkih, hranjenju in veterinarskih obiskih se sinhronizirajo:

  • Na zahtevo: Uporabnik osveži podatke ročno.
  • Samodejno: Strežnik periodično osvežuje podatke.
  • Ob spremembi: Podatki se sinhronizirajo ob vsaki pomembni posodobitvi.
